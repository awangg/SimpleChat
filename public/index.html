• Available Commands:
  • /help - Displays every command, its authorization level, and its effect
  • /list - Gets a list of users and corresponding ids
  • /setName [name] - Changes display name to the specified name
  • /emphasize [message] - Emphasizes the specified message
  • /getId [name] - Gets the id for the specified name
  • /getName [id] - Gets the name for the specified id
  • /mute [id] - Removes future messages from the specified id
  • /unmute [id] - Removes user from mute list
